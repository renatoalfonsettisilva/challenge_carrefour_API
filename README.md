API Tests for User Management

Este projeto contém testes automatizados para a API de gerenciamento de usuários, garantindo 100% de cobertura dos endpoints.

Endpoints Testados

- `GET /users`: Retorna uma lista de todos os usuários.
- `POST /users`: Cria um novo usuário.
- `GET /users/{id}`: Retorna os detalhes de um usuário específico.
- `PUT /users/{id}`: Atualiza as informações de um usuário.
- `DELETE /users/{id}`: Exclui um usuário.
